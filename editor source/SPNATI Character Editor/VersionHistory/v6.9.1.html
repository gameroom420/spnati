Metadata

  • Allows for the creation of futanari characters.
  • Supports showing a different layer count on the title screen (April Fool's characters only).
  • Supports characters with more than 8 layers (April Fool's characters only).

Dialogue

  • Futanari cases are now required.
  • The selector in the Collectible (Counter) condition only shows collectibles with counters.
  • The selector in the Collectible (Wearing) condition only shows wearable collectibles.

Collectibles

  • Allows linking a costume to a collectible, making the costume unlockable upon obtaining that collectible.

Pose Matrix

  • Prevents creating a Pose Matrix before defining the Wardrobe.

Epilogues

  • Sets a default value for every property of new sprites, instead of leaving them blank.

Validator

  • Adds hyphens to the list of acceptable characters to use in a marker name.

Miscellaneous

  • Adds support for KKL v107a4.
  • Supports the newly created "Broken" character status.
  • Increases the timeout for reading from KKL to 60 seconds.
  • Cleans up outdated code.

Bug Fixes

  • Only warns about custom settings not being available to a character if the character actually has custom settings.
  • Fixes older unofficial CE releases being always shown as "newer" than the current release.
  • Makes the eraser icon next to the crop values in the Pose Matrix actually work.
  • Fixes the clothing stage display in the Advanced Clothing Options dialogue.
  • Fixes the collectible selector not showing collectibles for other characters.
  • Fixes condition counts sometimes removing themselves when reopening a case.
  • Pose sets can now only be used in stages they are actually defined for.
  • Fixes some display issues for complicated marker conditions.
  • Generates the Intelligence table for a character if somehow none exists.
  • The Respond button no longer erases marker conditions that check another character's private marker.
  • No longer flags checks on other characters' private markers in the Validator.
