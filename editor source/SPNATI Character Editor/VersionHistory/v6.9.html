Dashboard

  • Displays the current and max number of Character Settings that are usable for this character.
  • Increases the file size limit to 20MB for characters who have passed sponsorship.

Wardrobe

  • Adds an Advanced Clothing Options dialog, for situations such as characters putting on more clothing in the middle of the game, or switching which clothing to take off mid-strip-sequence.

Dialogue

  • Adds a Nickname Operations tool to the advanced line settings.
  • Adds more color options for color-coding cases.
  • Prevents responding to the Opponent Deselected case as the target, which inherently makes no sense.

Pose Matrix

  • Adds buttons to reorder and rename pose sheets.

Pose Maker

  • Adds support for sprite clipping (cutting off the edges of the sprite).
  • Adds a toggle for turning off the box drawn around the currently highlighted sprite.
  • Allows use of @ and !@ operators in sprite conditions.

Pose Sets

  • Adds the ability to assign Pose Sets, a method to group together poses to more easily assign them to lines.

Epilogues

  • Adds an epilogue description field.
  • Adds support for sprite clipping (cutting off the edges of the sprite).
  • Adds a toggle for turning off the box drawn around the currently highlighted sprite.
  • Adds a toggle to show axes bisecting the scene.
  • Changes the default animation easing method to "Linear".
  • Allows adding a marker condition to an entire scene.
  • Directives and scenes can now be conditional on multiple markers.
  • Automatically renames an epilogue sprite if a sprite with the same name exists in the same scene.

Banter Wizard

  • Displays the name of the item being stripped and the stage number in the case description, instead of just "opponent_stripping".

Validator

  • Flags a lack of a description on the Metadata tab.
  • Flags markers with symbols other than letters, numbers, underscores, and plus signs.
  • Flags multiple Pose Maker sprites with the same ID.

Configuration

  • Adds support for per-character settings, which are options a player selects on the select screen that set markers.
  • Adds a Pose Preview Markers box, to enter which marker values the in-editor pose preview should treat as set when viewing Pose Maker poses. This can be toggled via the Preview Markers checkbox at the top of the Dialogue tab.

Miscellaneous

  • Adds F12 as a hotkey for the SFW mode, and allows toggling it from the File menu as well.
  • Allows processing of KKL codes larger than 32MB.
  • Some internal changes to the Respond function to produce better case conditions.

Bug Fixes

  • Fixes a major bug that sometimes modified the conditions of a line you were responding to.
  • Fixes crashes when attempting to open the Banter Wizard, Writing Aid, or Collectible Editor in SFW mode. (Collectible images are hidden when in SFW mode)
  • Fixes the bug causing costume dropdowns to not list costumes for specific characters.
  • Using the Configure Game feature no longer removes access to custom card decks.
  • The marker validation no longer erroneously flags some markers as never set.
  • Fixes an occasional crash when resizing the pose preview box.
  • Fixes some issues displaying ampersands in character names.
  • No longer automatically converts variable tests to lowercase, which broke case-sensitive tests.
  • Custom poses in costumes are no longer invisible in the image preview sidebar for stages that use the default folder.
  • collectibles.xml and tags.xml are correctly restored when reloading from backup.
